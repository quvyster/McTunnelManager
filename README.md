  McTunnelManager - Хости Minecraft без белого IP!
═══════════════════════════════════════════════════════════════

ЧТО ЭТО?
Программа, которая позволяет хостить Minecraft сервер дома даже 
если у тебя серый IP или CGNAT. Больше не нужен playit.gg!

КАК ЭТО РАБОТАЕТ?
Твой домашний ПК → SSH туннель → VPS с белым IP → Друзья играют!

═══════════════════════════════════════════════════════════════
  ЧТО НУЖНО
═══════════════════════════════════════════════════════════════

Дома (Windows):
• Windows 10/11
• 4+ ГБ оперативки (лучше 8 ГБ)
• Java 17+ (скачай на adoptium.net)
• PuTTY (для plink.exe - качай на putty.org)

VPS (аренда):
• Любой VPS с Ubuntu (от $2.50/мес)
• Белый IP адрес
• 512 МБ RAM достаточно

═══════════════════════════════════════════════════════════════
  БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════

ШАГ 1: Арендуй VPS
Зайди на Vultr/DigitalOcean/Hetzner, создай сервер Ubuntu.
Запомни IP адрес (например: 123.45.67.89)

ШАГ 2: Настрой VPS (один раз)
Подключись по SSH и выполни команды:

  sudo apt update && sudo apt upgrade -y
  sudo apt install openssh-server -y
  
  # Открой нужный порт
  sudo ufw allow 22
  sudo ufw allow 25565
  sudo ufw enable
  
  # Включи GatewayPorts (ВАЖНО!)
  sudo nano /etc/ssh/sshd_config
  
Найди строку "GatewayPorts" и напиши:
  GatewayPorts yes
  
Сохрани (Ctrl+O, Enter, Ctrl+X) и перезапусти SSH:
  sudo systemctl restart ssh

ШАГ 3: Создай SSH ключ (на Windows)
Открой PowerShell:

  ssh-keygen -t rsa -b 4096 -f C:\Users\ТВОЙ_ЛОГИН\.ssh\vps_key
  
Нажимай Enter на все вопросы.

Скопируй публичный ключ на VPS:
  Get-Content C:\Users\ТВОЙ_ЛОГИН\.ssh\vps_key.pub
  
Скопируй вывод (длинная строка начинается с ssh-rsa).

На VPS выполни:
  mkdir -p ~/.ssh
  nano ~/.ssh/authorized_keys
  
Вставь скопированный ключ, сохрани и закрой.

Установи права:
  chmod 600 ~/.ssh/authorized_keys

ШАГ 4: Конвертируй ключ в PPK
Открой PuTTYgen → Conversions → Import key
Выбери C:\Users\ТВОЙ_ЛОГИН\.ssh\vps_key
Нажми "Save private key" → сохрани как vps_key.ppk

ШАГ 5: Скачай Minecraft сервер
Скачай server.jar с minecraft.net
Положи в папку C:\MinecraftServer\
Запусти один раз, открой eula.txt
Измени eula=false на eula=true

ШАГ 6: Запусти McTunnelManager
Открой программу и заполни:

  VPS IP: 123.45.67.89 (твой IP VPS)
  SSH Port: 22
  SSH User: root
  Путь к plink.exe: C:\...\plink.exe
  Путь к SSH ключу: C:\...\vps_key.ppk
  
  Remote Port: 25565
  Local Port: 25565
  
  Тип сервера: Minecraft Java
  Путь к server.jar: C:\MinecraftServer\server.jar
  Память JVM: 2048 (2 ГБ)

Нажми "▶ СТАРТ ВСЁ"

ШАГ 7: Дай друзьям адрес
Скажи друзьям подключаться к: 123.45.67.89:25565

ВСЁ! Сервер работает 🎮

═══════════════════════════════════════════════════════════════
  ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
═══════════════════════════════════════════════════════════════

Туннель не запускается:
→ Проверь путь к plink.exe
→ Убедись что SSH ключ правильный
→ Проверь что VPS включен

Друзья не могут подключиться:
→ Проверь что GatewayPorts yes включен на VPS
→ Убедись что порт 25565 открыт в firewall
→ Проверь что туннель запущен (зелёная галочка)

Сервер не запускается:
→ Установи Java (java -version в PowerShell)
→ Проверь путь к server.jar
→ Измени eula.txt на eula=true
